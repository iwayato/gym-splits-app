# Gym Splits

App de rutinas de gimnasio (React + Vite + IndexedDB). Ver [gym_tracker_app_plan.md](gym_tracker_app_plan.md) para el plan original.

SPA 100% client-side: no hay backend, todos los datos se guardan localmente en IndexedDB; exportar/importar JSON para respaldo.

## Desarrollo

```bash
npm install
npm run dev      # servidor de desarrollo
npm run test     # tests de src/lib/stats.js (Vitest)
npm run build    # build de producción
```
